Notes:

  • Use keyword 'halt' in Auto Until to run until the program halts
  • Auto runs at about 3 million rules per second, expect large delays when running a large number of turns at once
  • The back button goes back exactly 1 million steps to unsure that memory limit is not exceeded
  • To save memory enable performance mode, this will disable updating the display and must be enabled before starting the machine

Example pattern: 1RB1LC_1RC1RB_1RD0LE_1LA1LD_---0LA
